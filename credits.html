Development

Main Programming & Concept: Shane Studios

RyxIDE aims to be a powerful, free, client-side IDE accessible to everyone.

Core Technologies & Libraries

  • Code Editor: Monaco Editor
  • Terminal Emulator UI: xterm.js
  • x86 Emulation: v86
  • AI Assistance: Google Gemini API
  • Markdown Rendering: Marked.js
  • Zip Functionality: JSZip
  • Icons: Font Awesome
  • (Legacy) Python Runtime: Pyodide
  • (Legacy) Ruby Runtime: ruby.wasm
